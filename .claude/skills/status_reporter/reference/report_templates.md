# Status Report Templates

## Daily Standup Template

### Format: What/Blocker/Next
- **What I accomplished yesterday:**
  - Task 1: Brief description of completed work
  - Task 2: Brief description of completed work
- **What I'm working on today:**
  - Task 1: Planned activities
  - Task 2: Planned activities
- **Blockers/Challenges:**
  - Issue 1: Description and impact
  - Issue 2: Description and impact

### Format: Accomplishments/Priorities/Risks
- **Accomplishments:**
  - Completed deliverable 1
  - Resolved issue 2
  - Achieved milestone 3
- **Today's Priorities:**
  - Priority 1: Description and expected outcome
  - Priority 2: Description and expected outcome
- **Risks & Concerns:**
  - Risk 1: Mitigation strategy
  - Risk 2: Mitigation strategy

## Weekly Status Report Template

### Executive Summary
- **Overall Status:** Green/Yellow/Red
- **Key Accomplishments:** Top 3 achievements
- **Critical Issues:** Top 3 challenges
- **Upcoming Priorities:** Next week's focus areas

### Detailed Status
- **Projects/Initiatives:**
  - Project Name: Status, Milestones, Issues, Next Steps
  - Project Name: Status, Milestones, Issues, Next Steps
- **Resource Utilization:** Team availability, capacity, allocation
- **Budget Status:** Spent vs. planned, variance analysis
- **Risk Register:** New risks, mitigation progress, residual risks

### Metrics & KPIs
- **Performance Indicators:** Key metrics with trends
- **Quality Metrics:** Defect rates, test coverage, customer satisfaction
- **Productivity Metrics:** Throughput, cycle time, efficiency ratios

## Monthly Status Report Template

### Strategic Overview
- **Mission Alignment:** Progress toward strategic objectives
- **Key Results:** Achievement of OKRs/KPIs
- **Market Position:** Competitive landscape changes
- **Organizational Health:** Culture, engagement, retention metrics

### Financial Summary
- **Revenue Performance:** Actual vs. forecast vs. prior period
- **Cost Management:** Budget utilization, cost trends, efficiency gains
- **Profitability Analysis:** Margins, cost structure, profit drivers
- **Investment Status:** Capital allocation, ROI analysis

### Operational Excellence
- **Process Improvements:** Efficiency gains, waste reduction
- **Quality Performance:** Defect trends, customer satisfaction
- **Innovation Metrics:** R&D investment, patent activity, new initiatives
- **Compliance Status:** Regulatory compliance, audit findings

## Quarterly Business Review Template

### Strategic Assessment
- **Vision Progress:** Movement toward long-term vision
- **Strategic Initiative Status:** Major program updates
- **Market Dynamics:** Industry trends, competitive positioning
- **Stakeholder Feedback:** Customer, partner, investor input

### Financial Performance
- **Revenue Analysis:** Growth trends, segment performance
- **Margin Analysis:** Cost structure, pricing effectiveness
- **Cash Flow:** Liquidity, investment capacity, debt management
- **Return Metrics:** ROIC, ROE, EBITDA margins

### Forward Outlook
- **Market Opportunities:** New segments, products, geographies
- **Strategic Priorities:** Next quarter's focus areas
- **Resource Planning:** Investment needs, hiring plans
- **Risk Assessment:** Emerging threats, mitigation strategies

## Project Status Report Template

### Project Overview
- **Project Name:** Current phase and overall status
- **Timeline:** Current milestone, critical path, schedule variance
- **Budget:** Spent vs. planned, cost variance, ETC/EAC
- **Scope:** Delivered vs. planned, change requests, scope creep

### Team Status
- **Resource Allocation:** Team composition, availability, utilization
- **Milestone Progress:** Completed, in-progress, upcoming
- **Deliverables:** Status, quality, acceptance criteria
- **Dependencies:** Internal, external, third-party

### Risk & Issue Management
- **Active Risks:** Probability, impact, mitigation status
- **Open Issues:** Description, owner, resolution date, impact
- **Escalations:** Issues requiring management attention
- **Lessons Learned:** Best practices, improvement opportunities

## Team Status Report Template

### Team Performance
- **Capacity Utilization:** Billable hours, project allocation
- **Skill Development:** Training, certifications, competency gaps
- **Collaboration:** Cross-team projects, knowledge sharing
- **Engagement:** Satisfaction, retention, feedback themes

### Delivery Metrics
- **Quality Indicators:** Defect rates, rework, customer feedback
- **Productivity Measures:** Throughput, velocity, efficiency
- **Client Satisfaction:** NPS scores, feedback themes, retention
- **Process Adherence:** SLA compliance, standardization

### Development Areas
- **Skills Gap Analysis:** Required vs. current capabilities
- **Process Improvements:** Bottlenecks, inefficiencies
- **Tool/Technology Needs:** Software, hardware, automation
- **Training Requirements:** Professional development needs

## Custom Report Elements

### Visual Components
- **Charts & Graphs:** Trend analysis, comparisons, forecasts
- **Dashboards:** Real-time metrics, KPI visualization
- **Heat Maps:** Performance by category, geography, segment
- **Timeline Views:** Project schedules, milestone tracking

### Narrative Sections
- **Storytelling:** Context, causation, implications
- **Comparative Analysis:** Benchmarking, peer comparison
- **Scenario Planning:** What-if analysis, sensitivity
- **Recommendations:** Action items, strategic advice

### Stakeholder-Specific Sections
- **Executive Level:** High-level summaries, strategic implications
- **Operational Level:** Detailed task status, resource allocation
- **Technical Level:** Architecture, implementation details
- **Financial Level:** Cost analysis, ROI calculations
